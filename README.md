# dev-bootstrap

Portable OpenCode configuration for Windows. One-command setup of plugin + MCPs + skills on any new machine.

See [ARCHITECTURE.md](ARCHITECTURE.md) for design, [docs/how-it-works.md](docs/how-it-works.md) for usage, [docs/troubleshooting.md](docs/troubleshooting.md) for common issues.

## Quick start

```powershell
git clone <repo-url>
cd dev-bootstrap
.\bootstrap\setup.ps1
```
